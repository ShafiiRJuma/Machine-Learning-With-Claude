# My Machine Learning Journey — Learning Python for Data Science with Claude as My Tutor

**Start date:** July 21, 2026
**Goal:** Go from complete beginner to job-ready in Python for Data Science and Machine Learning, one real-world scenario at a time.

---

## The Approach

Instead of following a textbook or a generic course, I'm learning through **scenario-based practice**, tutored by Claude. Every concept — from basic variables to Pandas to Matplotlib — is taught by building something real first, then getting graded and corrected on it. No passive lectures, no multiple choice quizzes: just build, get reviewed, fix bugs, move forward.

Each scenario is set in an East African / Tanzanian context (market ledgers, bajaji fares, mobile money, dala dala counts, clinic queues) so the practice feels grounded in something real rather than abstract "foo/bar" exercises.

The full curriculum is 28 scenarios across 5 modules:

| Module | Focus | Scenarios |
|---|---|---|
| 1 — [Python Fundamentals](https://github.com/ShafiiRJuma/Machine-Learning-With-Claude/tree/main/Python%20Fundamentals) | Variables, strings, conditions, loops, functions, dicts, file I/O, OOP | 01–08 |
| 2 — NumPy | Arrays, vectorized operations, aggregations, indexing, linear algebra basics | 09–13 |
| 3 — Pandas | DataFrames, cleaning, filtering, groupby, merging, datetime, EDA | 14–20 |
| 4 — Matplotlib & Seaborn | Charts, distributions, correlation, dashboards, styled reports | 21–25 |
| 5 — Capstones | Full pipelines, ML math from scratch, a real automation project | 26–28 |

---

## How Each Session Works

1. Show up, pick up exactly where I left off (no restarting from scratch unless I ask)
2. Get a scenario brief — real-world setting, raw inputs, requirements, expected output
3. Build the solution myself
4. Get graded (⭐ to ⭐⭐⭐⭐⭐) with bug-by-bug feedback and the "why," not just the "what"
5. Fix, resubmit if needed
6. Update the progress tracker below
7. Move to the next scenario

---

## What Comes After This

Once all 28 scenarios are mastered:
1. **ML Math in Python** — NumPy for calculus, linear algebra, and probability
2. **Scikit-learn** — training and evaluating real ML models
3. **Deep Learning** — PyTorch fundamentals
4. **Kaggle** — applying everything on real competition datasets
